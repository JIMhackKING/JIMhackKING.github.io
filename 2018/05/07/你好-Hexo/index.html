你好, Hexo!

2018-05-07

网上看到这个博客搭建平台很方便并且很强大，个人弄来试试

个人介绍

大家好，我是渣渣辉，哦不说错了，我是 DHC ,当然这只是我的网名，在这之前我一直想要自己用 Python 做出一个博客网站，尝试了 Django 和 Bottle，苦于不懂前端，于是一直做不出能见人的网站，今天偶然的机会让我遇到了 Hexo，百度了一下发现用他来搭建一个博客网站真的简单到不能简单了（喔真的不是卖广告的，是真的好用）我也推荐各位去尝试一下，做出属于自己的这么美丽的网站，当然还有很多地方都可以自己深度定制，我说了不算，你们用过才知道。

咳咳咳，是时候要找 Hexo 拿广告费了
